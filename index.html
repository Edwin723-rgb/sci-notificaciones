Acceso al sistema

Inicia sesión con la cuenta Google autorizada del despacho. El envío saldrá desde tu Gmail.

Paso 1: en Google Cloud → Credenciales crea un ID de cliente OAuth tipo "Aplicación web", activa en el mismo proyecto Gmail API, Google Calendar API y Google Drive API, y en "Orígenes JavaScript autorizados" agrega tu URL de Vercel y http://localhost. Copia el Client ID y pégalo arriba → Guardar.
Tras actualizar permisos (Calendar), usa Cerrar sesión y vuelve a entrar una vez para autorizar todo.
Paso 2: si la app está en modo prueba, añade como usuarios de prueba en la pantalla de consentimiento OAuth todos los correos con los que iniciarán sesión (incluye cuentas corporativas y personales como Hotmail si las usan Alejandra e Ingrid).
Opcional: también puedes pegar el ID en el código (window.__SCI_GOOGLE_CLIENT_ID__) o abrir la página con ?client_id=TU_ID.
